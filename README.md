A lightweight Windows utility to clean temporary files and optimise system performance.
This tool removes unnecessary files from common Windows locations and helps free up disk space.

## Features

* Cleans **User Temporary Files**
* Cleans **Windows Temp Folder**
* Clears **Prefetch Files**
* Empties **Recycle Bin**
* Displays **total files processed**
* Shows **disk space recovered**
* Lightweight and **portable**
* No installation required

## How to Use

1. Download the repository or click **Download ZIP** from GitHub.
2. Extract the ZIP file.
3. Open the extracted **device-optimiser** folder.
4. Double click **runOptimiser.bat**.

The optimisation process will start automatically.

## Requirements

* Windows 10 / Windows 11
* PowerShell enabled

## What the Tool Cleans

The script removes temporary files from:

* User Temp Folder
* Windows Temp Folder
* Windows Prefetch Folder
* Recycle Bin

This helps recover disk space and improve system performance.

## Safety

This tool only removes **temporary and unnecessary files**.
No system or personal files are modified.
